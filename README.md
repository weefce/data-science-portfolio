## Проекты 

### 1. [Прогнозирование оттока клиентов телеком-компании](01_customer_churn/)

**Задача:** Построить модель для предсказания ухода клиентов телеком-компании

**Что сделано:**
- Проведен EDA и feature engineering
- Созданы новые признаки на основе бизнес-логики
- Построены и сравнены модели: Logistic Regression, SVM, XGBoost
- Оптимизирован порог классификации под бизнес-задачу (recall 87%)
- Проанализирована важность признаков 

**Ключевые результаты:**
- Recall 87% - находим 87% уходящих клиентов
- Precision 50% - каждый второй предсказанный реально уйдет
- Топ-факторы оттока: временный клиент, помесячный контракт

**Технологии:** Python, Pandas, XGBoost, Scikit-learn

### 2. [Прогнозирование цены биткоина на 7 дней](02_bitcoin_forecasting/)

**Задача:** Сравнить статистические и ML модели для прогнозирования цен биткоина

**Что сделано:**
- Проведен анализ временных рядов (стационарность, ACF/PACF)
- Реализованы статистические модели: SARIMA, Exponential Smoothing, Prophet
- Построены ML модели с lag-признаками (XGBoost, линейные модели)
- Проведено строгое разделение данных для избежания утечки
- Сравнение всех подходов на разных горизонтах прогноза

**Ключевые результаты:**
- SARIMA модель: MAE = 328 (1.2% ошибки) - лучший результат
- Статистические модели превосходят naive baseline
- ML модели с lag признаками показали наихудшие результаты
- Выявлена высокая краткосрочная автокорреляция биткоина

**Технологии:** Python, Statsmodels, Prophet, XGBoost, Time Series Analysis
